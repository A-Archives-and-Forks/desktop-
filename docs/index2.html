TurboWarp Desktop

Use TurboWarp from an app on your desktop, even when you're offline.

TurboWarp is not affiliated with Scratch.

Changes since v0.0.4

  • Warp timer is now enabled by default. This may slightly reduce performance, but it greatly reduces the risk of crashes. It can be disabled in the Advanced menu.
  • The app now has an icon.
  • More parts of the app's interface are now translatable.
  • Various bug fixes and improvements.

Install TurboWarp Desktop for Windows

1
Download the installer.
2
Run the .exe file.
3
If a Windows SmartScreen alert appears, click "More info" then "Run anyways"

Install TurboWarp Desktop for macOS

1
Download the installer.
2
Open the .dmg file. Move TurboWarp Desktop into Applications.
3
Our binaries are not notarized, so additional steps are required. Open Finder, navigate to Applications, right click TurboWarp Desktop and select "Open" and then "Open" again.

Install TurboWarp Desktop with Snap

TurboWarp is published in the Snap store with the name turbowarp-desktop. Currently only x86 64-bit is supported.

1
Install snapd.
2
sudo snap install turbowarp-desktop

Install TurboWarp Desktop with AppImage

1
Download app. (x86 64-bit) Download app. (ARM 32-bit) Download app. (ARM 64-bit) Download app. (ARMv7) Alternative downloads for other architectures: Download the app for your system's architecture:
  • x86 64-bit
  • x86 32-bit
  • ARM 64-bit
  • ARMv7
2
Mark as executable. (typically a checkbox in right click > Properties > Permissions)
3
Run the app.
