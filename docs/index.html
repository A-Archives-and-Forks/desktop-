TurboWarp Desktop App

v0.0.1 - 2021-01-05

Use TurboWarp from an app on your desktop because that's something people asked for. It also has some addons.

Please remember to save your project often.

This is an early prototype. A lot of things don't quite work yet.

Change Username is broken.

It has that Scratch issue where "Load from your computer" is incredibly inconsistent.

Builtin sprite and sound libraries won't work offline.

Definitely some other things.

I'll try to fix these eventually -- it takes time.
