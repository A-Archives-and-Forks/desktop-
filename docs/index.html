TurboWarp Desktop

Use TurboWarp, the Scratch mod with a compiler, dark mode, addons, and more, from an app on your desktop. It even works when you're offline.

Install TurboWarp Desktop for Windows

  1. 1
    Download the installer.
  2. 2
    Run the .exe file.
  3. 3
    If a Windows SmartScreen alert appears, click "More info" then "Run anyways"

Install TurboWarp Desktop for macOS

  1. 1
    Download the installer.
  2. 2
    Open the .dmg file. Move TurboWarp Desktop into Applications.
  3. 3
    Our binaries are not notarized, so additional steps are required. Open Finder, navigate to Applications, right click TurboWarp Desktop and select "Open" and then "Open" again.

Automated installer

Run this in a terminal:

sudo bash -c "$(wget -qO- https://desktop.turbowarp.org/install.sh)"

Should work on any distribution and system. If it doesn't work, please let us know (include the full log). Alternative downloads are available below.

Is TurboWarp Desktop faster than regular TurboWarp?

Usually not, but it may be more stable as your computer won't unload it to save memory.
